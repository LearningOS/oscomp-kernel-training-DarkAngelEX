# FTL OS

FTL OS是2022全国大学生计算机系统能力大赛 - 操作系统赛- 内核实现赛道的参赛操作系统。FTL OS使用rust语言开发，基于rust async无栈协程进行上下文切换，能够充分使用hifive unmatched开发板资源。

## 目录

* 进程调度
  * 无栈协程
  * 任务调度器
  * 浮点数与优化
  * 进程管理 -- 未完成
  * 陷阱与上下文切换
* 内存管理
  * 页表映射
  * 映射管理器
  * 内存分配器
  * RCU系统
* 同步系统
  * 锁
  * 进程间通信
  * 信号系统 -- 未实现
* 文件系统
  * FAT32
  * 虚拟文件系统 -- 未实现
  * SD驱动
